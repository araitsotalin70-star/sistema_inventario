# Sistema de Gestión de Inventario y Pedidos

## Descripción

Este proyecto consiste en el desarrollo de un sistema básico de gestión de inventario y pedidos utilizando Python.
El programa permite registrar, mostrar, editar y eliminar productos y pedidos mediante el uso de listas (arreglos) y funciones, aplicando principios de programación estructurada.

El sistema se ejecuta desde la terminal y presenta un menú interactivo que facilita la administración del inventario y los pedidos.

---

## Objetivo del proyecto

Aplicar conceptos de programación como:

* Uso de arreglos (listas)
* Funciones y procedimientos
* Entrada y salida de datos
* Estructuras condicionales
* Ciclos o iteraciones
* Organización modular del código

---

## Funcionalidades

El sistema incluye las siguientes operaciones:

### Inventario

* Registrar nuevos productos
* Mostrar el listado de productos
* Editar productos existentes
* Eliminar productos del sistema

### Pedidos

* Registrar nuevos pedidos
* Mostrar pedidos registrados
* Eliminar pedidos
