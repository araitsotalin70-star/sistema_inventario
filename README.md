# Sistema de Inventario

## Descripción
Este proyecto implementa un sistema básico de gestión de inventario y pedidos utilizando Python.

El sistema permite registrar, mostrar, editar y eliminar productos y pedidos mediante un menú interactivo.

## Funcionalidades

- Registrar productos
- Mostrar lista de productos
- Editar productos
- Eliminar productos
- Registrar pedidos
- Mostrar pedidos

## Ejecución del programa

Ejecutar el programa con:

python sistema_pedidos.py
